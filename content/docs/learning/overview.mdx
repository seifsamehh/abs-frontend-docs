## üß† Logical Thinking & Problem Solving

### Essential Programming Problems

Enhance your logical thinking and problem-solving skills with these fundamental algorithmic challenges that every developer should master.

#### **Array & String Problems**

**Beginner Level:**

- **Two Sum** - Find two numbers that add up to a target
- **Valid Anagram** - Check if strings are anagrams
- **Palindrome Check** - Determine if a string reads the same forwards and backwards
- **Missing Number** - Find the missing number in a sequence
- **Maximum Subarray** (Kadane's Algorithm) - Find contiguous subarray with maximum sum

**Intermediate Level:**

- **3Sum** - Find three numbers that sum to zero
- **Longest Substring Without Repeating Characters**
- **Valid Parentheses** - Check if brackets are properly closed
- **Merge Intervals** - Combine overlapping intervals
- **Rotate Array** - Rotate array elements to the right by k steps

**Advanced Level:**

- **Sliding Window Maximum** - Find maximum in every window of size k
- **Minimum Window Substring** - Find minimum window containing all characters
- **Trapping Rain Water** - Calculate trapped rainwater
- **Median of Two Sorted Arrays** - Find median efficiently

#### **Tree & Graph Problems**

**Binary Tree Fundamentals:**

- **Tree Traversals** (Inorder, Preorder, Postorder)
- **Binary Tree Level Order Traversal**
- **Maximum Depth of Binary Tree**
- **Symmetric Tree** - Check if tree is mirror of itself
- **Path Sum** - Check if root-to-leaf path equals target

**Advanced Tree Problems:**

- **Lowest Common Ancestor** - Find LCA in binary tree
- **Serialize and Deserialize Binary Tree**
- **Binary Tree Maximum Path Sum**
- **Validate Binary Search Tree**

**Graph Algorithms:**

- **Depth-First Search (DFS)** and **Breadth-First Search (BFS)**
- **Number of Islands** - Count connected components
- **Clone Graph** - Deep copy a graph
- **Course Schedule** (Topological Sort)
- **Word Ladder** - Find shortest transformation sequence

#### **Dynamic Programming Classics**

**1D DP Problems:**

- **Climbing Stairs** - Count ways to reach the top
- **House Robber** - Maximum money without robbing adjacent houses
- **Coin Change** - Minimum coins to make amount
- **Longest Increasing Subsequence**
- **Word Break** - Check if string can be segmented

**2D DP Problems:**

- **Unique Paths** - Count paths in grid
- **Longest Common Subsequence**
- **Edit Distance** - Minimum operations to convert string
- **0/1 Knapsack** - Maximum value with weight constraint
- **Longest Palindromic Substring**

#### **Sorting & Searching**

**Essential Algorithms:**

- **Binary Search** and variations
- **Quick Sort** and **Merge Sort** implementations
- **Kth Largest Element** in array
- **Find First and Last Position** in sorted array
- **Search in Rotated Sorted Array**

#### **Backtracking & Recursion**

**Classic Problems:**

- **N-Queens** - Place queens on chessboard
- **Sudoku Solver** - Fill 9x9 grid
- **Permutations** and **Combinations**
- **Word Search** - Find word in 2D grid
- **Generate Parentheses** - All valid combinations

### üéØ Problem-Solving Strategy

#### **The UMPIRE Method**

1. **U**nderstand - Read problem carefully, identify inputs/outputs
2. **M**atch - Connect to known patterns/algorithms
3. **P**lan - Write pseudocode, consider edge cases
4. **I**mplement - Code the solution step by step
5. **R**eview - Test with examples, optimize
6. **E**valuate - Analyze time/space complexity

#### **Essential Problem Patterns**

**Two Pointers Technique:**

- Use when dealing with sorted arrays or palindromes
- Examples: Two Sum, Container With Most Water, Valid Palindrome

**Sliding Window:**

- For substring/subarray problems with optimization
- Examples: Longest Substring, Maximum Sum Subarray

**Fast & Slow Pointers:**

- Cycle detection in linked lists
- Examples: Linked List Cycle, Find Middle Node

**Merge Intervals:**

- Overlapping ranges problems
- Examples: Meeting Rooms, Insert Interval

**Cyclic Sort:**

- Array problems with numbers in specific range
- Examples: Find Missing Number, Find Duplicate

**Tree BFS/DFS:**

- Tree and graph traversal problems
- Examples: Level Order Traversal, Path Sum

**Binary Search:**

- Sorted array search and optimization problems
- Examples: Search Insert Position, Find Peak Element

### üèÖ Contest Preparation

#### **Competitive Programming Roadmap**

**Phase 1: Fundamentals (Months 1-3)**

- Master basic data structures (arrays, strings, stacks, queues)
- Learn fundamental algorithms (sorting, searching, basic math)
- Solve 200+ easy problems
- Participate in educational rounds

**Phase 2: Intermediate Skills (Months 4-8)**

- Advanced data structures (trees, graphs, heaps)
- Dynamic programming and greedy algorithms
- Number theory and combinatorics
- Aim for Codeforces rating 1200-1600

**Phase 3: Advanced Techniques (Months 9-18)**

- Complex algorithms (segment trees, network flows)
- Advanced math (modular arithmetic, probability)
- Contest strategies and time management
- Target rating 1600-2000+

#### **Contest Tips**

**During Contest:**

- Read all problems first (spend 5-10 minutes)
- Start with problems you're confident about
- Don't get stuck on one problem too long
- Submit as soon as you think solution is correct
- Keep track of penalty time

**Time Management:**

- Easy problems: 5-15 minutes
- Medium problems: 20-45 minutes
- Hard problems: 45-90 minutes
- Always leave time for testing
