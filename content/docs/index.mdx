## Our Tech Stack

We use modern, battle-tested technologies to build fast, scalable applications:

| Technology         | Version | Purpose                                     |
| ------------------ | ------- | ------------------------------------------- |
| **React**          | 19.x    | UI library with Server Components & Actions |
| **Next.js**        | 16.x    | Full-stack framework with App Router        |
| **TypeScript**     | 5.9.x   | Type-safe JavaScript                        |
| **Tailwind CSS**   | 4.x     | CSS-first utility framework                 |
| **TanStack Query** | 5.x     | Server state management & caching           |
| **Zustand**        | 5.x     | Simple client state management              |
| **shadcn/ui**      | Latest  | Accessible component library                |
| **Vitest**         | 2.x     | Unit & integration testing                  |
| **Playwright**     | 1.x     | E2E testing                                 |

[Learn more about our tech stack ‚Üí](/docs/reference/tech-stack)

## Development Principles

<Callout type="note">
  Our development approach is guided by these core principles:
</Callout>

- **Performance First** ‚Äî Build fast, responsive applications
- **Developer Experience** ‚Äî Tools and patterns that enhance productivity
- **Maintainability** ‚Äî Clean, readable, and well-tested code
- **Accessibility** ‚Äî Inclusive design for all users
- **Type Safety** ‚Äî Catch errors at compile time, not runtime

[Read our full philosophy ‚Üí](/docs/explanation/philosophy)
