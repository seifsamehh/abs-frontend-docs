## Our Tech Stack

We use modern, battle-tested technologies to build fast, scalable applications:

| Technology         | Version | Purpose                                       |
| ------------------ | ------- | --------------------------------------------- |
| **React**          | 19.x    | UI component library with concurrent features |
| **Next.js**        | 15.x    | Full-stack framework with App Router          |
| **TypeScript**     | 5.x     | Type-safe JavaScript                          |
| **Tailwind CSS**   | 4.x     | Utility-first styling                         |
| **TanStack Query** | 5.x     | Server state management                       |
| **Redux Toolkit**  | 2.x     | Client state management                       |
| **shadcn/ui**      | Latest  | Accessible component library                  |
| **Vitest**         | 3.x     | Unit testing                                  |
| **Playwright**     | 1.x     | E2E testing                                   |
