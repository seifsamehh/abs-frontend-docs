**Released:** December 20, 2024

#### Breaking Changes

- **React 19 Upgrade**: Updated to React 19 with new concurrent features

- **Tailwind CSS 4**: Migrated to Tailwind CSS v4 with new configuration format

<Tabs items={["Migration Guide", "What's New", "Deprecations"]}>
  <Tab value="Migration Guide">
    #### Migrating to React 19

    ```bash
    # Update dependencies
    npm install react@19 react-dom@19

    # Update types
    npm install -D @types/react@19 @types/react-dom@19
    ```

    **Key Changes:**
    - `useFormStatus` and `useFormState` are now stable
    - `use()` hook for reading resources in render
    - Ref as a prop (no more `forwardRef` needed in most cases)

    #### Migrating to Tailwind CSS 4

    ```css
    /* Old: tailwind.config.js */
    /* New: CSS-first configuration */

    @import "tailwindcss";

    @theme {
      --color-primary: #0ea5e9;
      --font-sans: "Inter", sans-serif;
    }
    ```
