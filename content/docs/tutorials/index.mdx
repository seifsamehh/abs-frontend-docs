## How to Use Tutorials

<Steps>
  <Step>
    ### Follow in Order Each tutorial builds on concepts from previous ones.
    Start from the beginning if you're new.
  </Step>
  <Step>
    ### Type the Code Don't copy-paste! Typing helps you learn and remember.
    Make mistakes - they're part of learning.
  </Step>
  <Step>
    ### Experiment After completing a tutorial, try modifying the code. What
    happens if you change something?
  </Step>
  <Step>
    ### Move to How-To Guides Once you're comfortable, check out [How-To
    Guides](/docs/how-to) for real-world problem solving.
  </Step>
</Steps>

## Prerequisites

Before starting tutorials, make sure you have:

- Node.js 18+ installed
- A code editor (we recommend VS Code)
- Basic JavaScript knowledge
- Your [development environment set up](/docs/how-to/setup/environment-setup)
