## React Learning Path

### Beginner (4-6 weeks)

**Week 1-2: JavaScript Fundamentals**

- ES6+ features (arrow functions, destructuring, spread)
- Promises and async/await
- Array methods (map, filter, reduce)

**Resources:** [javascript.info](https://javascript.info/), [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)

**Week 3-4: React Basics**

- Components and JSX
- Props and state
- Event handling
- Conditional rendering
- Lists and keys

**Resources:** [react.dev/learn](https://react.dev/learn)

**Week 5-6: React Hooks**

- useState and useEffect
- useRef and useCallback
- Custom hooks basics

**Project:** Build a todo app with local storage

### Intermediate (4-6 weeks)

**Week 1-2: Advanced State**

- useReducer for complex state
- Context API for global state
- State management patterns

**Week 3-4: Performance**

- React.memo and useMemo
- useCallback optimization
- Code splitting with lazy/Suspense
- React DevTools profiling

**Week 5-6: Testing**

- React Testing Library
- Testing hooks
- Mocking and async tests

**Project:** Build a dashboard with data fetching, filtering, and pagination

### Advanced (4+ weeks)

**Week 1-2: Architecture**

- Component composition patterns
- Render props and HOCs
- Compound components
- Controlled vs uncontrolled

**Week 3-4: Advanced Patterns**

- Error boundaries
- Portals and refs
- Server components concepts
- Concurrent features

**Ongoing: Deep Dives**

- React internals (fiber, reconciliation)
- Custom renderers
- Contributing to React ecosystem

**Project:** Build a design system with Storybook

## TypeScript Learning Path

### Beginner (2-3 weeks)

**Core Concepts:**

- Basic types (string, number, boolean, array)
- Type inference
- Interfaces vs types
- Function typing

**Resources:** [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/), [Total TypeScript Beginners](https://www.totaltypescript.com/tutorials/beginners-typescript)

### Intermediate (2-3 weeks)

**Advanced Types:**

- Generics
- Union and intersection types
- Type guards and narrowing
- Mapped and conditional types
- Template literal types

**Resources:** [TypeScript Deep Dive](https://basarat.gitbook.io/typescript/)

### Advanced (Ongoing)

**Mastery Topics:**

- Variance (covariance/contravariance)
- Module augmentation
- Declaration files
- Performance optimization
- Complex generic patterns

**Resources:** [Type Challenges](https://github.com/type-challenges/type-challenges), [Total TypeScript Advanced](https://www.totaltypescript.com/)

## Next.js Learning Path

### Beginner (2-3 weeks)

**Core Concepts:**

- App Router basics
- Pages and layouts
- Navigation and linking
- Static vs dynamic rendering
- Data fetching basics

**Resources:** [Next.js Learn Course](https://nextjs.org/learn), [Next.js Documentation](https://nextjs.org/docs)

### Intermediate (3-4 weeks)

**Advanced Routing:**

- Route groups
- Parallel routes
- Intercepting routes
- Middleware

**Data Patterns:**

- Server Actions
- Revalidation strategies
- Streaming and Suspense
- Error handling

**Project:** Build a full CRUD app with authentication

### Advanced (Ongoing)

**Production Patterns:**

- Advanced caching strategies
- Performance optimization
- Monitoring and analytics
- Custom servers
- Deployment strategies

**Project:** Build a multi-tenant SaaS application
