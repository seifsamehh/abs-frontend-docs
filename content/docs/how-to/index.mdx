## How-To vs Tutorial

| How-To Guides            | Tutorials                |
| ------------------------ | ------------------------ |
| Solve a specific problem | Learn a skill            |
| Assume basic knowledge   | Teach from scratch       |
| Jump to what you need    | Follow start to finish   |
| "How do I configure X?"  | "Let's build X together" |

<Callout type="tip">
  **New to something?** Start with a [Tutorial](/docs/tutorials) first, then
  come back here when you need to solve specific problems.
</Callout>
